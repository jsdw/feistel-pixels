About

A Feistel cipher provides a one-to-one mapping from some input range to some output range. With the right settings, the output values given back can appear to be random. This page takes a number of samples (starting from input = 0 and working up) and plots them as black pixels onto a canvas, so that we can get a feel for the level of randomness for different settings.
